# Тестовое задание для компании «Создаватель»

**Позиция:** Web-разработчик (Backend)
**Стек:** PHP, 1C-Bitrix, JavaScript

Репозиторий содержит решения тестового задания. Мой основной стек - PHP и фреймворк Laravel. Это мой первый опыт работы с CMS 1С-Битрикс. В процессе выполнения задания я самостоятельно развернул BitrixVM, разобрался с базовой архитектурой системы, инфоблоками, механизмом событий и структурой компонентов.

## Структура репозитория

Код разделен на директории согласно блокам исходного задания:

* `/block-1-bitrix` — Решения для 1С-Битрикс. Здесь находится папка `local` с кастомным компонентом и бизнес-логикой (`php_interface/init.php`). Описание настроек инфоблоков и логики находится во внутреннем `README.md`.
* `/block-2-frontend` — Решения по верстке и нативному JavaScript.
* `/block-3-php` — Скрипты на чистом PHP.

## Окружение и запуск

* **Версия PHP:** 8.4
* **Редакция 1С-Битрикс:** Редакция "Бизнес" на базе официальной виртуальной машины BitrixVM.
* **Запуск Блока 3**: Скрипты на чистом PHP можно проверить автономно. Для этого необходимо запустить встроенный PHP-сервер, открыв консоль в папке `block-3-php` и выполнив команду:
`php -S localhost:8000`
После этого открыть в браузере нужный скрипт (например, http://localhost:8000/task-3.4-gallery/).
* Пояснения по настройке инфоблоков и логике Битрикс-заданий находятся в файле `README.md` внутри папки `/block-1-bitrix`.
